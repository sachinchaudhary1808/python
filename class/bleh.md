
# if 
### this statement is used to run the block of code based on true or flase values returned by conditional statement, if it is true it will execute block of code comes btwif and else otherwise block given after else.

# if else if
### this statement is used to run the block of code by checking multipal conditions or multipal conditional statement and execute the block of code by conditions,

```python
if condition :
    ...# block of code
elif condition :
    ...# block of code
    .
    .
    .
else :
    ...# block of code
```


### nested if
    when we use if condition block inside another if it is known as nested if.


```python
if condition :
    #block of code
    if condition :
        #block of code
    else :
        #block of code 
    #block of code 
else :
    #block of code
```
